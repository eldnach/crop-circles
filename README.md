# Procedural VFX Demo 

Compute, Indirect Rendering and GPU Culling used to render large amounts of procedural geometry with low rendering overhead. 

Procedural drawing also reduces build size, by eliminating the need for prebaked assets. The demo uses a single 256x256 noise texture:
